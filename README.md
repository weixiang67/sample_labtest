← branch→
Git branch //check which branch u at
Git branch “bug-fix1” // a new branch called bug-fix1, cannot have space between
Git checkout “bug-fix1” // this changes your directory to “bug-fix1”
Git branch // you will now see two branches

After all the edits in the branch, 
Git add, git commit
Git checkout “master” //back to the main branch
Then git merge “bug-fix1”, then merge the branch u made :)
